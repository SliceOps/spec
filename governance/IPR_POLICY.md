## SliceOps and reference runtimes

SliceOps is an open methodology. Vendor runtime implementations (including the first reference runtime) are architectural peers — none is "the official" runtime, and the methodology never requires a specific runtime. See `../DISCLOSURE.md`.

## Contributions
